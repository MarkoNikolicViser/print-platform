# print-platform
This platform is a web and mobile based printing marketplace that connects users with local print shops. Users upload documents, choose a print shop, pay online, and get notified when their files are ready for pickup. Print shops manage prices and orders through a dedicated dashboard.

Requirements
* node -v 20+

Project Start
* cd front
* yarn dev
* cd back
* cd strapi
* yarn
* yarn dev

Login params for the dummy account
* admin@copyshop.rs
* admin123
